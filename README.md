# BOT-GPT-WA
Bot-GPT-Wa adalah project bot whatsapp dengan mengkoneksikan API OpenAI dengan whatsapp menggunakan library baileys

## Installation
```bash
git clone https://github.com/4nndmw/bot-gpt-wa
cd bot-gpt-wa
npm install
npm run
```
